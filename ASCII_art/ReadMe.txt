		ASCII art generator

Convert your image (.png/.jpg/.jpeg) to text-image (.txt) 

How to use: 1) Make sure you have installed all necessary python libraries - see below
	    2) Run "main.py" in the ASCII_art folder
	    3) Insert the URL of your image in the entry or click "Select picture" to choose your image from your PC respect (.png/.jpg/.jpeg) format
	    4) Set your settings - download option and resolution of generated text-image (download to file Results-Výsledky)
	    5) Generate
	    6) Click "Copy" to copy text-image to the clipboard or click "New image" to generate a new image or identical one with a different size or click "Exit" to exit the program

Tips: You can achieve the best text image by setting the resolution as close as possible to the original resolution of image


Python libraries: Have Python installed on your PC, for installing libraries open CMD and insert a line with pip
		  - Customtkinter - https://pypi.org/project/customtkinter
        		pip install customtkinter
		  - Pillow - https://pypi.org/project/Pillow
        		pip install Pillow
		  - Requests - https://pypi.org/project/requests
        		pip install requests

		ASCII art generátor

Převeďte svůj obrázek (.png/.jpg/.jpeg) na textový obrázek (.txt)

Jak používat: 1) Ujistěte se, že máte nainstalovány všechny potřebné python knihovny – viz. níže
	      2) Spusťte "main.py" ve složce ASCII_art
	      3) Vložte URL svého obrázku do vstupu pro text nebo klikněte na "Vybrat obrázek" a vyberte obrázek z vašeho PC ve formátu (.png/.jpg/.jpeg)
	      4) Zvolte si nastavení - možnost stažení a velikost vygenerovaného text-obrázku (stažení do file Results-Výsledky)
	      5) Generovat
	      6) Klikněte na "Kopírovat" pro zkopírování textového obrázku do schránky nebo klikněte na "Nový obrázek" pro vygenerování nového obrázku nebo identického obrázku s jinou velikostí nebo klikněte na "Exit" pro ukončení programu

Typy: Nejlepšího textového obrázku dosáhnete nastavení rozlišení co nejvíc shodné z původním rozlišením obrázku


Knihovny Pythonu: Mějte na svém počítači nainstalovaný Python, pro instalaci knihoven otevřete CMD a vložte řádek s pip
		  - Customtkinter - https://pypi.org/project/customtkinter
         		pip install customtkinter
		  - Pillow - https://pypi.org/project/Pillow
         		pip install Pillow
		  - Requests - https://pypi.org/project/requests
         		pip install requests
